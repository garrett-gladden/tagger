# Teams Room Dashboard Site

This project is a static HTML page that presents a dashboard for monitoring Teams Room device performance. It is built using HTML, CSS, and JavaScript, and is designed to be hosted on GitHub Pages.

## Project Structure

```
teams-room-dashboard-site
├── index.html         # Main HTML document
├── css                # Directory for CSS files
│   └── styles.css     # Styles for the HTML page
├── js                 # Directory for JavaScript files
│   └── main.js        # JavaScript code for interactivity
├── src                # Directory for source files
│   └── teams-room-dashboard.jsx # Original React component code
├── .gitignore         # Git ignore file
├── CNAME              # Custom domain for GitHub Pages
└── README.md          # Project documentation
```

## Setup Instructions

1. **Clone the Repository**: 
   Clone the repository to your local machine using:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**: 
   Change into the project directory:
   ```
   cd teams-room-dashboard-site
   ```

3. **Open `index.html`**: 
   Open `index.html` in your web browser to view the dashboard.

## Usage

- The dashboard provides real-time monitoring and analytics for Teams Room devices.
- Use the filters to customize the view based on country, city, manufacturer, device type, and subnet.

## Publishing to GitHub Pages

To publish the page to a publicly accessible GitHub Pages site, follow these steps:

1. **Create a New Repository**: 
   Go to GitHub and create a new repository.

2. **Push the Project Files**: 
   Add the remote repository and push the files:
   ```
   git remote add origin <repository-url>
   git push -u origin main
   ```

3. **Enable GitHub Pages**: 
   In the repository settings, enable GitHub Pages and select the main branch or root directory as the source.
